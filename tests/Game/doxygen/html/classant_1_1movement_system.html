Detailed Description

Sistema encargado de hacer que las entidades se muevan.

Las entidades se moveran si tienen el componente Velocity y Transform. Se moveran segun la rotacion que tengan y la velocidad actual. Cuando recibe Eventos tipo COLLISION_EVENT parara a las entidades si se estan moviendo.
