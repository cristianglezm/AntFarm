Crea un nivel segun la imagen especificada.

La imagen especificada tiene que tener:

Colores

  1. Color blanco es ignorado.
  2. Color negro es suelo.
  3. Color rojo es la puerta de entrada al nivel.
  4. Color verde es la puerta de salida del nivel.
Parameters
namestd::string nombre del nivel.
backgroundstd::string nombre de la textura del fondo.
lvlsf::Image imagen a cargar con la informacion del nivel.
nEntitiesint numero de entidades que tendra el nivel.
overTimesf::Time tiempo maximo para pasarse el nivel.
boundssf::FloatRect rectangulo con los limites del mundo(Tienen que ser iguales a la imagenes de fondo.)
Returns
std::unique_ptr<World> World creado
