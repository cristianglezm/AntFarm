Detailed Description

Sistema encargado de hacer que las entidades se muevan.

Las entidades se moveran si tienen el componente Velocity y Transform. Se moveran segun la rotacion que tengan y la velocidad actual. Maneja a las entidades segun los eventos que reciba del sistema de colision.

Eventos que maneja:

  1. TERRAIN_COLLISION: cambiara la rotacion de las entidades
  2. COLLISION_EVENT: ~
