Detailed Description

Sistema que comprueba las colisiones entre entidades.

Cuando detecta una colision añade un evento a la EventQueue con las dos entidades que colisionaron.

  1. Event<Entity*,Entity*> -> type: COLLISION_EVENT
  2. Event<Entity*,int> -> type: TERRAIN_COLLISION
  3. Event<Entity*> -> type: OUT_MAP
